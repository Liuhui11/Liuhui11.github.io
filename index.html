js中的浅拷贝和深拷贝

发表于 2017-09-24 | 分类于 js

举个例子来说明一下什么是浅拷贝什么是深拷贝

浅拷贝

得出的结果可以看出是浅拷贝

  • 非对象的属性值一个改变不影响另一个的值
  • 对象属性是引用赋值所以一个改变会影响另一个的改变

出现这种情况的本质是：对象是按引用赋值的

深拷贝

指的是拷贝一个对象，改变一个值不影响另一个的值

一个技巧

使用JSON对象的parse()和stringify()方法

  • pares() JSON字符串转成js对象
  • stringify() 将js对象转成JSON字符串

这种方法有局限性：

  • 对于正则表达式类型、函数类型等无法进行深拷贝(而且会直接丢失相应的值)
  • 它会抛弃对象的constructor，不管这个对象原来的构造函数是什么，在深拷贝之后都会变成Object

打包：将应用程序统一放在一个后缀是ipa的文件中，然后发给其他人，可以安装在手机上供用户或测试人员安装

可安装ipa的前提：

  • 说清楚是哪一个应用程序（App Id）
  • 可以安装在哪一台设备上（UDID）

打包过程：

  1. 在本地生成csr文件
  2. 进入开发者中心 apple developer
  3. 选择Ad Hoc生成一个iOS_distribution.cer证书，此证书是为了限定哪台电脑可以打包生成ipa
  4. 新建一个App Id，此ID用于限定要打包的应用程序的Bundle Id是什么，这次不能写*这样的范围匹配，必须写一个准确的App Id
  5. 注册要安装ipa文件的设备的UDID
  6. 选择Ad Hoc，利用iOS_distribution.cer文件 + 设备UDID + App Id生成一个provision文件
  7. 将证书安装到电脑上
  8. 保证xcode的配置一致
  9. 将ipa拷贝给要安装的人，此人打开iTunes，点击应用程序，再双击ipa，点击安装，点击同步即可

手动编写一个ajax,不依赖第三方库 理解每一步

ajax请求

一个完整的ajax请求一般包括以下步骤：

  • 实例化XMLHttpRequest对象
  • 连接服务器
  • 发送请求
  • 接收响应数据

http相关状态码

  • 200 ok 访问正常
  • 301 Moved Permanently 永久移动
  • 302 Move Temporarity 暂时移动
  • 304 Not Modified 未修改
  • 307 Temporary Redirect 暂时重定向
  • 401 Unauthorized 未授权
  • 403 Forbidden 禁止访问
  • 404 Not Found 未发现指定网址
  • 500 Internal Server Error 服务期发生错误

解决跨域

跨域的几种实现方式（ajax不能跨域）

  1. 通过跨域标签
  • script标签
  • link标签
  • img标签 图片防盗链，知乎做了
  1. 服务器端设置http header
  • 另一个解决跨域的简洁方法,需要服务器端来做
  • 但是作为交互房,我们必须知道这个方法
  • 是将来解决跨域问题的一个趋势

    什么是跨域?

    浏览器有同源策略,不允许ajax访问其他域的接口 接口的安全性(人家的接口怎么能让你随便访问的)

    跨域条件

    协议 域名 端口 有一个不同就算跨域

jsonp实现原理

请求时服务器可以返回动态的内容 然后拼接成js文件通过script标签引入

请求过程:
  • 请求阶段: 浏览器创建一个script标签,并给其src赋值(类似http://example.com/api/?callback=jsonpCallback)
  • 发送请求: 当给script的s4c赋值时,浏览器会发起一个请求
  • 数据响应: 服务器将要返回的数据作为参数和函数名称拼接在一起(格式类似”jsonpCallback({name: ‘abc’})”)返回.
  • 当浏览器接受到了响应数据,由于发起请求的是script,所以相当于直接调用jsonpCallback方法,并且传入了一个参数 why?

原型和原型链

发表于 2017-07-15 | 分类于 js

原型的5个规则

- 所有的引用类型（数组、对象、函数）都具有对象特性，即可自由扩展属性（除了“null”意外）
- 所有的引用类型（数组、对象、函数）都有一个\__proto__（隐式原型）属性，属性值是一个普通的对象
- 所有的函数都有一个prototype（显示原型）属性，属性值也是一个普通的对象
- 所有的引用类型（数组、对象、函数），\__proto__属性值指向它的构造函数的"prototype"属性值
- 当试图得到一个对象的某个属性时，如果这个对象本身没有这个属性，那么会去他的\__proto__（即它的构造函数的prototype）中寻找

补充两点：

- this永远指向实例本身
- 循环对象自身的属性

js日期格式转换

发表于 2017-06-29 | 分类于 js

请实现一个dateFormat方法，实现将时间戳按照固定格式进行转换

  • dateFormat(‘1350052653’, ‘YYYY-MM-DD HH:MM:SS’); // 1970-01-16 23:00:52
  • dateFormat(‘1350052653’, ‘YYYY-MM-DD’); // 1970-01-16

格式化参数不传时默认为’YYYY-MM-DD HH:MM:SS’，年月日的分隔符必须为’-‘，时分秒的分隔符必须为’:’，该参数支持任意组合，例如：’YYYY-MM HH:MM’、’YYYY-MM’等，但不会出现’YYYY-DD’或’HH:SS’此类情况

原型和原型链

发表于 2017-07-15 | 分类于 js

原型的5个规则

- 所有的引用类型（数组、对象、函数）都具有对象特性，即可自由扩展属性（除了“null”意外）
- 所有的引用类型（数组、对象、函数）都有一个\__proto__（隐式原型）属性，属性值是一个普通的对象
- 所有的函数都有一个prototype（显示原型）属性，属性值也是一个普通的对象
- 所有的引用类型（数组、对象、函数），\__proto__属性值指向它的构造函数的"prototype"属性值
- 当试图得到一个对象的某个属性时，如果这个对象本身没有这个属性，那么会去他的\__proto__（即它的构造函数的prototype）中寻找

补充两点：

- this永远指向实例本身
- 循环对象自身的属性

js日期格式转换

发表于 2017-06-29 | 分类于 js

请实现一个dateFormat方法，实现将时间戳按照固定格式进行转换

  • dateFormat(‘1350052653’, ‘YYYY-MM-DD HH:MM:SS’); // 1970-01-16 23:00:52
  • dateFormat(‘1350052653’, ‘YYYY-MM-DD’); // 1970-01-16

格式化参数不传时默认为’YYYY-MM-DD HH:MM:SS’，年月日的分隔符必须为’-‘，时分秒的分隔符必须为’:’，该参数支持任意组合，例如：’YYYY-MM HH:MM’、’YYYY-MM’等，但不会出现’YYYY-DD’或’HH:SS’此类情况
